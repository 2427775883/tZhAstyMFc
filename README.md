# 项目简介

## 核心功能
- 登录注册
- 收入支出记录
- 图表统计
- 收入记账管理
- 支出记账管理
- 收入支出统计
- 个人信息修改

## 适用场景
- 个人财务管理
- 家庭账目管理
- 零散收入支出记录

## 技术特点
- 使用Python编程语言
- 基于 Django 框架开发
- 数据存储于 MySQL 数据库
- 前端界面采用 HTML 技术
